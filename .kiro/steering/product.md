# Cloud Optimization Assistant (COA)

A comprehensive AWS security analysis and cloud optimization platform that combines Amazon Bedrock agents with Model Context Protocol (MCP) servers to provide intelligent security assessments and recommendations.

## Core Components

- **Web Chatbot Interface**: Real-time chat interface with Claude 3.5 Sonnet for natural language interactions
- **Amazon Bedrock Agent**: AI agent orchestrating security analysis workflows with multi-MCP integration
- **MCP Server Architecture**: Specialized tools for AWS security assessments via Model Context Protocol
- **Cross-Account Integration**: Secure, read-only access patterns for multi-account security analysis
- **Centralized Authentication**: Amazon Cognito providing unified authentication across all components

## Primary Use Cases

- Comprehensive AWS security posture evaluation
- Well-Architected Framework security pillar assessments
- Real-time security findings analysis and prioritization
- Cross-account security analysis with least-privilege access
- Interactive security recommendations and remediation guidance

## Current Status

- **Active**: Security assessment capabilities with 6 specialized MCP tools
- **Planned**: Cost optimization, reliability assessment, performance analysis, and operational excellence evaluation
